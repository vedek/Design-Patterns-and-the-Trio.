The Generator Pattern

The generator pattern is a code generator and data generator pattern, especially relevant in the present Java release, in conjunction to the Eclipse Collections, based on GS Collections. AIML is adopted to Java and JSON based per-observer pattern. The JSON generator pattern uses AIML and the state pattern with a library of functions and functoids and factoids, HOF and functors to enhance JSON or YAML generators. Integration with parser pattern, use of Jison and yoeman are also mentioned.
